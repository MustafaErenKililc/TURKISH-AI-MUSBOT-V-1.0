## 🌍 About the Project
MUSBOT AI is a global artificial intelligence assistant developed by **Mustafa Eren Kilic**. This project aims to bring the power of Gemini AI to a clean, terminal-style web interface. It's designed to be fast, reliable, and user-friendly.

## 🔥 Key Features
- **Global Intelligence:** Powered by Google Gemini API.
- **Cyber Interface:** High-tech neon green terminal design.
- **Standalone App:** Can be added to the mobile home screen as a Web-App.
- **No Hardcoded Keys:** Users provide their own API keys for maximum security.
